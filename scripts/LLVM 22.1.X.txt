I am new to describing actions, **so if you are too, I don't recommend gathering it yourself**. It's **better** to download from the official repo https://github.com/DinoMax0n/WinClang. If you encounter any errors, write to me at dinomax0n@gmail.com
(I changed the scripts and now there are only the most basic things to work with, everything that is intuitively understandable for an experienced person, I deleted)

The scripts are designed assuming that the host compiler is located in C:/PROGRA~2/LLVM and that the new compiler will be installed in the C:/PROGRA~2/LLVMNew directory

Prerequisites:

Windows10/11 
MSYS2-UCRT64 (For MinGW-UCRT)
Ninja
Cmake
swig
git
Clang/GCC
lld (There may be bugs during linking with GNU-ld)
Python (I recommend 3.14)

EXECUTE THE COMMANDS EXACTLY IN THE SEQUENCE IN WHICH THEY ARE WRITTEN!

(Tested 22.1.0,22.1.1,22.1.2,22.1.3)

RUN THE FOLLOWING COMMANDS IN PowerShell ONLY(or change the installation prefix)

Then execute all commands in MSYS2-UCRT64 WITH ADMIN rights (or change the installation prefix)
git clone https://git.code.sf.net/p/mingw-w64/mingw-w64

../mingw-w64-headers/configure \
  --host=x86_64-w64-mingw32 \
  --prefix=/ \
  --enable-sdk=all \
  --enable-idl \
  --enable-secure-api \
  --with-default-win32-winnt=0x0A00 \
  --with-default-msvcrt=ucrt

make install DESTDIR="/c/PROGRA~2/LLVM/x86_64-w64-mingw32"

(The command can freeze for a couple of minutes even on powerful hardware)
../mingw-w64-crt/configure \
    --host=x86_64-w64-mingw32 \
    --with-default-msvcrt=ucrt \
    --disable-lib32 \
    --prefix=/ \
    CC="C:/PROGRA~2/LLVM/bin/clang.exe" \
    CXX="C:/PROGRA~2/LLVM/bin/clang++.exe" \
    CFLAGS="-target x86_64-w64-mingw32 -m64 -Wno-typedef-redefinition" \
    CXXFLAGS="-target x86_64-w64-mingw32 -m64 -Wno-typedef-redefinition" \
    --with-sysroot=/c/PROGRA~2/LLVM/x86_64-w64-mingw32
	
make

The command is dangerous because you have to write the path manually (to avoid accidentally running the command in the wrong place and breaking something)

Get-ChildItem -Path "PrefixInstallLLVM\lib\clang\22\lib\x86_64-w64-windows-gnu" -Filter "*-x86_64*" | Rename-Item -NewName { $_.Name -replace '-x86_64', '' }
